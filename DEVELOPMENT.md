# mbed Cloud SDK for Python

## Prerequisites

Python 2.6+ / Python 3.3+

## Installing

After cloning this repository, run:

```bash
> python setup.py develop
```
## Tests

1. Clone [mbed-cloud-sdk-testrunner](https://github.com/ARMmbed/mbed-cloud-sdk-testrunner)
2. See example usage
